先天性脑积水

又称为婴儿脑积水，胚胎期或婴幼儿期脑脊液产生吸收间的失衡和（或）脑脊液循环受阻致脑室扩大，颅腔因颅缝未闭而代偿性扩大。

分类

  • 梗阻性脑积水：系脑室系统狭窄部位梗阻
    室间孔 中脑导水管 第四脑室开口处 侧孔和基地池
  • 交通性脑积水: 脑脊液不能被蛛网膜颗粒吸收所产生的脑积水

病因

  1. 脑脊液产生过多

  2. 脑脊液吸收障碍

    常见脑膜炎，蛛网膜下腔出血后发生蛛网膜下腔粘连，静脉窦血栓形成上腔静脉综合征等是常见原因

  3. 脑脊液循环受阻

临床表现

  1. 颅压增高引起的症状
    头痛，呕吐，视神经乳头水肿
  2. 头围和头部形态异常
    头围增长，前卤增大膨隆，“落日征”
  3. 神经功能障碍
    展神经麻痹造成斜眼，复视。常以斜视就诊眼科
  4. 静水期脑积水
    脑积水进展到一定程度后趋于平衡 也称“代偿性脑积水”

辅助检查

  1. 腰椎穿刺
    测定颅内压
  2. x线检查
    颅骨变薄，骨缝增宽
  3. CT
    安全快捷显示脑室扩张部位和程度
  4. MRI
    较CT敏感，准确显示脑室和蛛网膜下腔形态，大小和狭窄部位。脑脊液动力学检查动态了解脑脊液循环状况
  5. 超声
    床旁超神动态监测新生儿脑室形态和脑室内出血

治疗

  1. 非手术治疗
    暂时性措施，药物治疗包括乙酰唑胺，脱水剂，利尿剂和抑制脑脊液分泌药物
  2. 手术治疗
    1. 脑室-腹腔分流术
    2. 脑室-右心房分流术
    3. 腰大池腹腔分流术
    4. 神经内镜下三脑室造瘘术
  3. 手术后并发症
    1. 穿刺并发症
    2. 分流管梗阻
    3. 感染
    4. 分流管位移
    5. 过度引流
    6. 裂缝脑室综合征
