下载nodejs

==npm== -v查看node版本

安装hexo博客框架

借助npm包管理工具下载淘宝镜像源npm install -g cnpm registry=https://registry.npm.taobao.org 全局安装
即安装cnpm工具
这里新建了一个blog的文件夹以下操作在这个文件夹下完成

mkdir blog
cd blog

cnpm install -g hexo-cli 安装hexo框架
初始化hexo

hexo init

新建文章

hexo new”name”

生成(generate)静态文件

hexo g

清理

hexo clean

hexo 部署到git上的工具hexo-deployer-git

cnpm install –save hexo hexo-deployer-git

建gihub仓库

在github上新建一个名为‘yourname.github.io’的仓库
然后在blog中找到_config_yml的配置文件在deploy下配置

deploy:
type: git
repo: 仓库的地址

然后清理和部署

hexo clean
hexo d
